Lives (3) - Each player has 3 lives to make it through a level. Wrong answer results in -1 life. If you lose all 3 lives you fail the level. 

Each round of quiz lasts 180 seconds. Each question will have 4 choices, one of which will be correct. 

Enemy hp bar will have 100 HP. Each correct answer will -5 from the enemy hp. -100 beats the level. If they beat the boss and still have time left, they can get bonus pearls. See [[Pearls]]
